# Ethiopia Home Find

## Overview

Ethiopia Home Find is a React + Vite application for browsing and listing rental houses across Ethiopian cities. It includes an owner flow for submitting listings, an admin dashboard for approvals and moderation, and user browsing with filtering.

## Getting Started

```sh
git clone <REPO_URL>
cd ethiopia-home-find
npm install
npm run dev
```

## Tech Stack

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
